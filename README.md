3. Задача: найти утерянные данные. На вход поступают 2 массива чисел одного рамера. Программа должна определить зависисмость между двумя массивами. Зависимость будет исключительно линейная. На вход будет поступать новое число из массива x, на выходе - новое из y.
например:
x = [1, 2, 3, 4, 5]
y = [1, 4, 6, 8, 10]

вход: 6
выход: 12
